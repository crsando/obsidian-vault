# 🌙 伪街「夜行挑战」任务清单 · 上海篇

> 专为夜间设计，避开商场，人少但不荒凉，循序渐进地接触人群。  
> 适合上海 20:00 以后出门的姐妹～

* * *

## 🌟 Lv.1 — 夜色掩护（心跳指数 ★☆☆☆☆）

几乎不需要和人打交道，先让自己习惯"穿着出门"这件事。

- [x] **小区花园散步 10 分钟** — 最安全的起点，楼下转一圈就回来，重点是迈出那一步
- [x] **在自动贩卖机买一瓶水** — 上海遍地都是，全家门口那种就行，零社交
- [ ] **沿苏州河/滨江步道走一段** — 晚上散步的人不少但没人在意你，灯光也好看
- [x] ~~**骑共享单车兜一圈** — 速度快看不清脸，风吹头发的感觉很爽~~
- [ ] **在路边椅子上坐 5 分钟刷手机** — 练习"在公共空间自然地存在"
- [ ] **戴着耳机经过一组跳广场舞的阿姨** — 她们根本不会看你，但你会心跳，信我

## 💗 Lv.2 — 擦肩而过（心跳指数 ★★☆☆☆）

开始有短暂的人际接触，但都是一次性的、可控的。

- [x] **便利店（全家/罗森）买一瓶饮料** — 上海便利店店员见多识广，没人大惊小怪
- [ ] **在路边摊/煎饼摊买个夜宵** — "老板来个煎饼加蛋"，3 秒交互搞定
- [ ] **丰巢/菜鸟驿站取快递** — 输入取件码就行，驿站灯光暗，压力很小
- [ ] ~~**扫码开一辆哈啰单车骑去 2 公里外的便利店** — 组合任务！骑车+买东西~~
- [x] **经过保安亭时跟保安点个头** — 微小的社交，但意义重大
- [x] **在无人超市/自助结账买东西** — 盒马之类的自助结账通道，全程不用说话
- [ ] ~~**拿着手机假装在导航，问路人"请问 XX 路怎么走"** — 第一次主动搭话！~~

## 💖 Lv.3 — 融入夜色（心跳指数 ★★★☆☆）

需要停留更久、交流更多，但环境依然友好。

- [ ] **去 24 小时药店买点东西** — 维生素、创可贴都行，药店店员很专业不会多看
- [ ] **在社区超市（好德/可的）买 5 样东西** — 拎着购物篮逛一圈，排队结账
- [ ] **坐在街边奶茶店等一杯外带** — 点单+等待，大概要站 3-5 分钟
- [ ] **去小区门口的菜鸟驿站寄一个快递** — 需要报手机号、填地址，交流更多
- [ ] **在肯德基/麦当劳的自助点餐机点一份外带** — 有人排在你后面也别慌
- [x] **打一辆出租车（短途）** — 告诉师傅目的地，上海出租车司机阅人无数
- [x] **去连锁药店买一盒套** — 经典任务夜间版！晚上店里人少，压力减半
- [ ] **便利店买东西时跟店员说"帮我加热一下"** — 多一句话的交流

## 💝 Lv.4 — 夜间漫步（心跳指数 ★★★★☆）

在户外停留更久，路线更长，逐步建立"我属于这里"的感觉。

- [ ] **沿安福路/武康路夜间散步 30 分钟** — 法租界的梧桐树下，灯光温柔，文艺青年多，包容度高
- [ ] **在街边花店买一束花** — 老板会跟你聊几句，"送人还是自己养？""自己的～"
- [ ] **去深夜书店坐一会儿（如大隐书局/钟书阁）** — 安静环境长时间停留
- [ ] **在社区健身步道跑步/快走 20 分钟** — 运动装也可以很可爱
- [ ] **用饿了么跑腿让骑手送东西给你** — 在楼下接个外卖，短暂对视
- [ ] **路过水果摊买点水果，跟老板砍砍价** — "草莓怎么卖的？便宜点嘛～"
- [ ] **去全家打印一张文件** — 需要跟店员有来回沟通
- [ ] **在公园长椅上坐着吃一个冰淇淋** — 慢慢吃，享受夜风，你很好看
- [ ] **经过酒吧街（如永康路）不进去，只是走过** — 感受一下自己在热闹人群中的存在感

## 👑 Lv.5 — 夜之女王（心跳指数 ★★★★★）

你已经可以自在地穿梭在夜色里了。

- [ ] **一个人坐在街边吃一碗馄饨/面** — 路边小店，坐下来，慢慢吃完
- [ ] **在便利店买东西时被店员叫"小姐姐"** — 不是你能控制的，但当它发生时，你赢了
- [ ] **规划一条 1 小时的伪街夜行路线并走完** — 从家出发，经过至少 3 个打卡点
- [ ] **去照相馆/地铁站的自助拍照机拍一张照片** — 留下纪念！
- [ ] **夜间独自坐一站地铁** — 上海地铁末班车前人不多，2 号线除外
- [ ] **去小区门口的社区食堂/夜市吃个晚饭** — 长时间坐在公共区域
- [ ] **在外面待满 2 小时，完成至少 3 个任务后回家** — 综合大挑战！

* * *

## 🗺️ 上海推荐夜行路线

### 路线 A：法租界文艺线

武康路 → 安福路 → 乌鲁木齐中路 → 五原路

> 梧桐树 + 咖啡馆灯光 + 年轻人多 + 非常包容

### 路线 B：滨江放松线

徐汇滨江步道 / 杨浦滨江

> 开阔、人不多不少、灯光好、适合散步和骑车

### 路线 C：老城厢烟火线

打浦路 → 日月光附近 → 斜土路

> 生活气息浓，小吃摊、水果摊、药店密集，适合做任务

### 路线 D：大学城周边线

松江大学城 / 五角场附近

> 学生多、年轻、对多元外表接受度极高

## ⚠️ 安全守则

- **时间控制** — 建议 20:00-23:00 之间，太晚人太少不安全
- **路线报备** — 跟朋友分享实时位置（微信"共享实时位置"）
- **避开偏僻路段** — 有灯光、有监控、有行人的路才走
- **带好证件** — 晚上偶尔有巡逻查身份证，不要慌，正常配合就好
- **卸妆预案** — 包里放卸妆湿巾，随时可以切换状态
- **交通** — 优先用滴滴/出租车回家，别走夜路太久
- **天气** — 上海梅雨季/台风天就别出门了，等好天气再战
- **穿平底鞋** — 夜间路况不如白天，安全第一

* * *
