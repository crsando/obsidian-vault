# 🎀 伪街「心跳挑战」任务清单

# 🎀 伪街「心跳挑战」任务清单

由浅入深，分级递进，每完成一个打个 ✅ 给自己攒勇气值！

* * *

## 🌟 Lv.1 — 新手试水（心跳指数 ★☆☆☆☆）

- [x] **便利店买水** — 最基础的出门任务，进去、拿瓶水、结账、走人。感受一下"我真的出门了"的感觉
- [x] **戴耳机在小区散步一圈** — 晚上8点以后，光线暗，人少，先适应走路姿态
- [x] **自动贩卖机买饮料** — 零社交压力，但你确实在户外了！

## 💗 Lv.2 — 开始社交（心跳指数 ★★☆☆☆）

- [ ] **超市购物满5件** — 需要在货架间逛一会儿，会被人看到，练习淡定
- [ ] **奶茶店点单** — 需要开口说话！点个可爱的饮品名更加分
- [ ] **取快递** — 对着快递员说自己的名字/取件码，短暂交流
- [ ] **药店买创可贴** — 正常到不能再正常的事，但就是心跳加速

## 💖 Lv.3 — 信心建立（心跳指数 ★★★☆☆）

- [ ] **去超市买套** — 经典任务！结账时面不改色就算过关
- [ ] **商场逛服装店** — 在女装区自然地翻看衣服，假装在挑选（其实是真的在挑）
- [ ] **坐一趟公交/地铁** — 在封闭空间里和陌生人共处，全程保持气场
- [ ] **麦当劳/KFC柜台点餐坐下吃** — 需要停留较长时间的公共场所

## 💝 Lv.4 — 进阶挑战（心跳指数 ★★★★☆）

- [ ] **化妆品柜台试色** — 让柜姐帮你试口红，大方地说"我想看看这个色号"
- [ ] **拍大头贴/自助照相机** — 留下伪街出行的纪念！
- [ ] **咖啡店坐一下午** — 带本书或电脑，长时间在公共场合自然地存在
- [ ] **请路人帮忙拍照** — 需要主动搭话，心跳拉满

## 👑 Lv.5 — 传说级（心跳指数 ★★★★★）

- [ ] **商场试衣间试穿** — 拿几件衣服进去试，出来问店员"好看吗"
- [ ] **和朋友逛街吃饭** — 以女装状态赴一次正式社交
- [ ] **去漫展/同好聚会** — 在最友好的环境里大方展示自己

* * *

## ⚠️ 安全提示（很重要！）

- **结伴出行** — 前几次最好有朋友陪同/在附近待命
- **选择友好区域** — 大学城、商业区、年轻人多的地方通常更包容
- **新手先走夜场** — 光线暗容忍度高，等自信了再挑战白天
- **带好卸妆用品** — 随时可以有退路，心理压力会小很多
- **注意人身安全** — 偏僻的地方不要去，太晚了早点回家

> 核心原则：**开心最重要，量力而行，每一步都是进步！** 没有人规定必须从 Lv.1 开始，也没有人催你通关。按自己的节奏来就好～ 🌈
