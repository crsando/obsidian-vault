## 一句话核心

市场把 **中国化学（601117）** 当作**建筑工程股**（个位数 PE、常年破净附近）定价，但公司实际在往**化工新材料实业**转型。**这个"身份错配"就是最大的预期差**——下有工程主业和高分红兜底，上有**己二腈国产替代**的期权价值。

## 一、两条腿走路

### 1. 传统工程主业（基本盘 / 安全垫）

- 全球**化工工程龙头**，**现代煤化工**设计施工国内第一梯队，这是核心护城河。
- 新签合同、在手订单充足，**海外占比持续提升**（中东、俄罗斯、中亚"一带一路"），且海外毛利率明显高于国内。
- 驱动力：**油价、中东资本开支、现代煤化工景气度**。
- 属性：低毛利、高周转、重现金流——典型建筑央企，估值天然被压低。

### 2. 实业转型（弹性 / 想象空间）

- **天辰齐翔己二腈**是皇冠明珠。己二腈是**尼龙66**的核心原料，此前被英威达 / 巴斯夫 / 奥升德三家海外巨头垄断，**天辰实现工艺国产化突破**，战略意义 >> 短期利润意义。
- 其他新材料矩阵：气凝胶、PBAT 可降解塑料、己内酰胺、DMC 等。
- 逻辑升级：从"帮别人建厂" → **"自己建厂卖产品"**，多赚**制造 + 周期**的钱。

## 二、三个层次拆解定价

| 层次 | 内容 | 你在为什么付钱 |
|---|---|---|
| **下有底** | 破净附近、低 PE、股息率不错、央企中特估 / 市值管理 | 安全垫 |

## 三、必须盯的关键变量

- **己二腈 / 尼龙66 价格与价差**——直接决定实业利润弹性，是周期品，近年行业新增产能多、价格偏弱。
- 天辰齐翔项目**产能利用率、成本 / 良率、二期进展**。
- **新签合同增速**（尤其海外那块）。
- 现代煤化工政策与景气度。
- **应收账款 / 现金流质量**（建筑央企通病，回款是命门）。
- 化工投资的资本开支对当期利润的拖累 + 减值风险。
- 分红率变化。

## 四、主要风险

- 实业是**重资产 + 周期品**，**己二腈价格持续低迷时，新材料会从"利润贡献"变成"利润拖累"**——这是近年市场最担心的点。
- 传统工程受固投 / 地产拖累，回款风险。
- 海外地缘、汇率风险。
- 转型资本开支大，自由现金流承压。

## 五、核心矛盾（也是 alpha 来源）

**市场用建筑股估值，公司想讲成长股故事。** 预期差存在，但兑现节奏取决于两件事：**己二腈周期反转** + **放量兑现**。周期不反转，故事就一直是故事。

# 中国化学 × 油价：分层拆解

关系**不是单向的**：不同业务板块方向不一样、时滞也不一样。简单说"油价涨 = 利好"是偷懒。

## 传导路径一：工程主业 ← 油价（正相关，但慢，是领先指标）

**油价高 → 资源国 + 煤化工资本开支扩张 → 新签合同增长**

- 油价高 → 中东、俄罗斯、中亚**产油国财政宽裕** → 加大石化 / 化工项目投资 → 中国化学海外订单（高毛利那块）增加。
- 油价高 → **现代煤化工经济性提升**（煤制油、煤制烯烃相对原油有成本优势）→ 项目审批和投资意愿上升 → 直接砸在中国化学最强的煤化工工程能力上。

**关键：这是慢变量。** 油价上涨 → 资本开支决策 → 招标 → 中标 → 确认收入，链条走完通常要 **1~3 年**。所以**油价对它是领先景气指标，不是当期利润驱动**。

## 传导路径二：实业化工 ← 油价（方向不定，看价差不看绝对值）

己二腈 / 尼龙66、己内酰胺本质是**石油基产品**（往上追是丁二烯、苯）：

- 油价高 → **原料成本抬升**（利空成本端）。
- 同时油价高 → **整个石化链价格中枢上移**（利好售价端）。
- 净影响 = 看**价差（产品价 − 原料成本）**能否扩大，以及公司**向下游转嫁**的能力。

所以对实业，**单纯油价涨 ≠ 利好**。近年己二腈价差被压缩，主因是产能过剩（供需），跟油价关系不大。**盯价差，别盯油价绝对值**。

## 核心洞察：时滞 + 方向的错配

| 板块 | 对油价 | 方向 | 时滞 |
|---|---|---|---|
| 工程订单 | 领先指标 | **正相关** | 慢（1~3年） |
| 实业利润 | 成本 + 售价 | **方向不定** | 快（当期） |

**同一个油价变动，在订单端是慢正相关，在实业利润端是当期、方向不定**——两者会在不同时间尺度上打架。所以**不能简单给它贴"油价受益股"标签**。

## 放到周期股框架里看

中国化学**不是纯周期股**：

- 工程主业是**"后周期 / 资本开支跟随"**属性，是**卖铲子的**，滞后于油价，对油价的 beta 远低于荣盛、恒力那种一体化炼化。
- 只有实业那小块才带点顺周期商品味道。
- 更像**"油气 / 化工 capex 产业链的滞后承包商"**，而不是油价的直接映射。

## 实操结论

看油价对它的意义，应看：**"油价中枢能否持续支撑资源国 + 煤化工的资本开支"**——这是订单的源头。而不是盯短期油价日线波动去交易它。**短期油价跳动对当期业绩几乎没影响，真正的传导藏在 1~3 年后的订单里**。
