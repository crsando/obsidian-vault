# A股存储/内存半导体板块深度研报（截至2026年7月14日）

## 一、总体概况：全球存储超级周期，A股存储板块集体狂飙

全球存储行业正处于分析师所称的 **"超级周期"**——AI算力需求爆发（单台AI服务器DRAM搭载量为传统服务器的8-10倍，NAND用量达3倍），叠加供给端结构性紧缺，高盛称市场正面临 **15年来最严重的存储芯片供应短缺**。2026年全球存储产值预计突破 **5500亿美元**，首次超过晶圆代工规模，成为半导体产业第一增长极。

**A股半导体板块过去一年（2025.7-2026.7）**总市值从约5.6万亿增长超10.3万亿，区间平均涨幅 **183%**，远超大盘。2026年6月，申万半导体指数单月上涨 **36.75%**，估值处于2019年以来 **99.72%分位**。

## 五、中美科技博弈对存储板块的影响

| 时间 | 事件 | 影响 |
|---|---|---|
| 2022年10月 | 美国对华存储设备出口管制升级 | 长鑫/长江存储设备获取受限，倒逼国产设备加速 |
| 2024年 | 长江存储被列入1260H军事企业清单 | 增加融资和国际合作不确定性 |
| 2025年初 | 长鑫存储被追加列入1260H清单 | 同上 |
| **2026年2月14日** | **美国防部将长江存储、长鑫存储移出1260H清单** | 阶段性利好，但实体清单约束仍在 |
| 2026年2月17日 | 美国拟禁止联邦机构采购含长鑫/长江存储/中芯国际芯片的产品 | 政府采购层面新限制 |
| 总体态势 | 长江存储仍在实体清单，设备/材料获取存合规风险；但两家企业已实现规模化量产，国产设备替代加速 |

**核心结论**：制裁客观上加速了国产替代进程。长鑫和长江存储已从"被制裁压制"转向"制裁环境下仍能高速增长"，4000亿元级累计投资开始兑现。

## 八、关键结论

1. **业绩层面**：A股存储公司正经历有史以来最强业绩周期——兆易创新利润增1099%，长鑫科技单季赚330亿，江波龙利润暴增6万倍+。
2. **估值层面**：板块估值已处于历史极端分位（99.72%），兆易创新PE 70x+、PB 16x+，远超海外同业，市场高度亢奋但分歧加大。
3. **产业层面**：中国存储首次形成DRAM+NAND"双龙头"格局（长鑫+长江存储），从"0到1"进入"1到N"的规模化阶段。
4. **风险点**：周期性本质未变、估值透支、中美制裁不确定性、全球存储厂商可能重启扩产压力。
5. **催化剂**：长鑫科技7月16日申购 → 有望成为A股首只真正的存储IDM标的，万亿市值期待。
