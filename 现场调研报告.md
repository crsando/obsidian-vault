# 整体感受

- 董事长谭总：具有典型理工男的特征，展现出对细节的把控和行业专业性
	- 特点：
		- 向投资者展现了宏伟的战略蓝图：先于有关政府部门，在地方政府的支持下，抢先发展卫星激光“骨干网”的建设，明确放弃to-C，针对to-B和to-G的大客户业务（典型应用案例：将电价低的新疆等地IDC数据机房，Token工厂，眉山数字太空港等，链接到市场需求区域，代替传统的光纤实现低的延迟和更好的经济性）。这种“骨干网”的战略和市场定位，也是该公司商业模式区别与其他卫星通讯公司（可能聚焦于物联网等）的主要区别，是目前市场内唯一如此定位的。
		- 对于对未来5-10年的计数发展和迭代有自己的判断和认识，认为接下来的激光通讯的数据带宽/吞吐量将从现在1.6T进一步提升到6.4T，22T，往更高速率推进，对技术发展表现出乐观的态度和激情。
		- 对于技术类、生产工艺类问题，能展开来详细的解读和展示，尤其是在提问中涉及到的生产工艺优化和降本增效的话题时，能娓娓道来，具体到如“z轴应力”，“点贴”等工艺细节和生产制造中的具体问题等等，对生产制造展现出自身的见解，也认为工业化生产能力是自身以及该公司区别于传统“航天人”（工艺）的重要方面。
		- 语言表达上具有理工男直爽的特点，在定位上表达了客观冷静的政治立场和对社会当前利益分配现状的判断，讲红色政治，积极拥护政府的领导。
		- 在沟通中表达了对于国内卫星激光领域学术研究和主要相关方的全面了解，但
		- 似乎表现出对佛教的偏好和认同，办公室墙上挂着“法华寺”求来的墨宝（内容：骐骥驰骋），有表示想用卫星发送“经文上天”等，提到公司内部有专人负责文旅相关的合作与发展。
	- 关注：
		- 表现出对行业主要参与者，技术大佬的熟悉程度，但经常想不起人名（理科男特征）
		- 坚持正确的立场与站位，但在一些具体语言表达上略显随意，如部分设计政府市场利益分配的问题，也包括较为直接的向与会的一名投资者表示“你看太多科幻了“等。
		- 对于尽调报告中的一些关注问题，如“收入确定”等，被提问后并没有给出具体的措施或行动，回答较为笼统
	- 补充：
		- 面相富态，短发，黑色半框眼镜，上衣为带长江商学院logo的POLO衫，白色球鞋非常干净，着装风格符合理工男的一致特征。
- 团队人员
	- 董事长办公室宽敞，紧邻副总“罗总”的办公室，均在办公楼二次，走访中穿过二层办公区，肉眼估计大概60-80个工位，座位有人率约70%，不排除部分人员走动或在另一边测试间等。
	- 工位为常规的Cube，大部分工位都是双显示器，打开画面基本是3D设计界面等
	- 有一位副总合同到期不续：系准备回母校交大教书。离职会自动放弃所有的股票，谭总解释说是因为该副总原先为网络方向，后因公司发展需要导致被迫转型为硬件方向，自身对公司贡献有限也因方向切换问题不是特别舒适，加之家境较好（据说老婆家房地产开放商，很多套房），因此即便放弃上市股票也要回交大教书）
	- 团队核心人员基本都是谭总的吉林大学校友，整体非常年轻，目测都在30岁上下
	- 据董秘介绍：目前人才管理分为两条线，一边是研发团队，一边是生产团队。据谭总介绍：从业内的主要研究机构（成都光电所）挖来了大量研发人才。 
- 现场调研中的一些重要主题
	- 战略发展目标
		- 自建星座，建设中国的卫星通讯“骨干网”，目标是25000亿（5000颗卫星，每颗5000万）
		- 滚动投资，类似Starlink一样，不会一次性5000颗，而此逐步滚动的发射上线部署，预计从72颗星开始实现盈利，从120颗星开始实现盈亏平衡。
		- 表面看投资额很高，但资金主要是依赖地方国企，企业本身其实并非”重资产“，而是技术牵头；真正操盘的是无锡国资委，加三大运营商/IDC，常州国资委等。
		- “骨干网”的商业定位和技术路线和其他比如“智慧车联”，“物联网”等完全不同，是唯一依赖“高速设备”的定位，所以实际上是国内唯一一家的定位。
		- 合作的都是大厂，大客户，如杭州阿里的之江研究所，在与阿里等合作沟通中直接向吴泳铭汇报，阿里方面非常支持
		- （建成后）主要挣钱来源是数据公司（如阿里，腾讯，字节），帮助他们用更低延迟的卫星激光通讯替代传统的光纤通讯，实现更低的延迟，帮助他们实现在低电价地区（如新疆阿勒泰）建设数据中心仍能保证业务（如大模型推理）的响应速度。
		- 不to-C（商业模式和需要的技术方案不同，所以该公司与其他商业航天/卫星通讯公司不存在实质性的竞争）
		- 不做军工：控制风险
		- 积极出海：正在沟通新加坡，阿布扎比等境外项目的落地。在境外，相比Starlink，我们的优势是更能平衡多方的利益需要。
