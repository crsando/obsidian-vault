# 整体感受

- 董事长谭总：具有典型理工男的特征，展现出对细节的把控和行业专业性
	- 特点：
		- 向投资者展现了宏伟的战略蓝图：先于有关政府部门，在地方政府的支持下，抢先发展卫星激光“骨干网”的建设，明确放弃to-C，针对to-B和to-G的大客户业务（典型应用案例：将电价低的新疆等地IDC数据机房，Token工厂，眉山数字太空港等，链接到市场需求区域，代替传统的光纤实现低的延迟和更好的经济性）。这种“骨干网”的战略和市场定位，也是该公司商业模式区别与其他卫星通讯公司（可能聚焦于物联网等）的主要区别，是目前市场内唯一如此定位的。
		- 对于对未来5-10年的计数发展和迭代有自己的判断和认识，认为接下来的激光通讯的数据带宽/吞吐量将从现在1.6T进一步提升到6.4T，22T，往更高速率推进，对技术发展表现出乐观的态度和激情。
		- 对于技术类、生产工艺类问题，能展开来详细的解读和展示，尤其是在提问中涉及到的生产工艺优化和降本增效的话题时，能娓娓道来，具体到如“z轴应力”，“点贴”等工艺细节和生产制造中的具体问题等等，对生产制造展现出自身的见解，也认为工业化生产能力是自身以及该公司区别于传统“航天人”（工艺）的重要方面。
		- 语言表达上具有理工男直爽的特点，在定位上表达了客观冷静的政治立场和对社会当前利益分配现状的判断，积极拥护政府的领导，但
		- 在沟通中表达了对于国内卫星激光领域学术研究和主要相关方的全面了解，但
		- 似乎表现出对佛教的偏好和认同，办公室墙上挂着“法华寺”求来的墨宝，有表示想用卫星播放经文等，提到公司内部有专人负责文旅相关的合作与发展。
	- 关注：
		- 坚持正确的立场与站位，但在一些具体语言表达上略显随意，如部分设计政府市场利益分配的问题，也包括较为直接的向与会的一名投资者表示“你看太多科幻了“等。
		- 对于尽调报告中的一些关注问题，如“收入确定”等，被提问后并没有给出详细的回答，
