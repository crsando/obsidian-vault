# 达叔的完整护肤计划（含水杨酸 + 壬二酸）

达叔的完整护肤计划（含水杨酸 + 壬二酸）🌅 早间流程（按顺序）步骤产品用法① 清洁清水 or 芙丽芳丝氨基酸洁面（T区油时用）温水 ~35°C，不搓揉② 壬二酸壬二酸凝胶（15%或20%），局部点涂鼻头痘痘/痘印薄薄一层，等吸收③ 化妆水MUJI 敏感肌用化妆水拍在脸上，趁皮肤微湿④ 乳液MUJI 敏感肌用乳液全脸涂匀⑤ 防晒理肤泉大哥大（通勤）/ 安热沙小金瓶（户外）一元硬币量，出门前15分钟涂🌙 晚间流程（按顺序）步骤产品用法① 清洁芙丽芳丝氨基酸洁面泡沫打圈，洗掉防晒和油脂② 水杨酸Stridex 棉片（红罐2%）擦拭鼻头及T区，停留不用洗③ 等待—等 2-3 分钟让水杨酸吸收干爽④ 化妆水MUJI 敏感肌用化妆水拍全脸⑤ 精华雅诗兰黛小棕瓶（选用）水之后乳之前，一滴管⑥ 乳液MUJI 敏感肌用乳液锁水，全脸涂匀📅 一周节奏安排周一周二周三周四周五周六周日早：壬二酸✅✅✅✅✅✅休晚：水杨酸✅休✅休✅休休⚠️ 注意事项水杨酸一周 3 次就够，鼻头耐受之后可以慢慢加到隔天一次，别急着天天用壬二酸可以每天早上用，它本身比较温和，负责抗炎退红淡痘印周日给皮肤放假，两样都停一天，只做基础清洁 + 保湿 + 防晒刷酸期间防晒必须到位，角质变薄后不防晒等于白搞如果出现脱皮、刺痛，水杨酸先减到一周两次，壬二酸也减量顺序原则：先薄后厚、先活性成分后保湿锁水📝 产品清单汇总洁面：芙丽芳丝 (Freeplus) 净润洗面霜水杨酸：Stridex 施颜适红罐棉片（2%）壬二酸：壬二酸凝胶（15%-20%）化妆水：MUJI 敏感肌用化妆水乳液：MUJI 敏感肌用乳液精华：雅诗兰黛第七代小棕瓶（晚间选用）防晒（通勤）：理肤泉 (La Roche-Posay) 大哥大防晒乳防晒（户外）：安热沙 (Anessa) 小金瓶更新于 2026-05-05，综合原有护肤指南 + 水杨酸/壬二酸痤疮治疗方案v2：水乳更换为 MUJI 敏感肌系列🦾 额外局部护理项目产品用法手肘保湿尿素霜早晚各一次，涂抹手肘干燥粗糙处尿素霜软化角质 + 深层保湿，专治手肘这种容易被忽略的粗糙暗沉区域，坚持用会明显变滑。
