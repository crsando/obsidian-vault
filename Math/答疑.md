### 0

因式分解

$$

    (a+b+c+d)(b+c-a-d)(c+a-b-d)(a+b-c-d) + 16 abcd

$$

***Solution***

这道题的关键是注意到$d$的“定位”和其他三个变量$a,b,c$不同，而$a,b,c$是轮换对称的。

为了看着舒服一点，我们结合本题的结构，这么换元

$$
\begin{aligned}
	x &= b + c - a\\
	y &= c + a - b \\
	z &= a + b - c
\end{aligned}
$$

则$a+b+c = x+y+z$，以及$a = \frac{y+z}{2}$（其他依次类推），故

$$
\begin{aligned}
    & (a+b+c+d)(b+c-a-d)(c+a-b-d)(a+b-c-d) + 16 abcd \\
    = & (d + x + y + z)(x - d)(y - d)(z - d) + 2d(x+y)(y+z)(z+x) \\
    = & - (d + x + y + z)(d - x)(d - y)(d - z) + 2d(x+y)(y+z)(z+x) \\
\end{aligned}
$$

这个式子直接展开有点复杂，但是容易看出来$d = x + y + z$的时候原式$=0$，所以原式必有因式$(x + y + z - d)$，同理，$d = - x$, $d = -y$, $d = -z$的时候，原式$=0$，故原式必有因式$(d+x)(d+y)(d+z)$。


由于原式是$x,y,z,d$是四次齐次，比较两边的系数，必然存在某个待定系数$k$，使得以下等式恒成立

$$
\begin{aligned}
    & (a+b+c+d)(b+c-a-d)(c+a-b-d)(a+b-c-d) + 16 abcd \\
    = & (d + x + y + z)(x - d)(y - d)(z - d) + 2d(x+y)(y+z)(z+x) \\
    = & k(x +y + z - d)(x + d)(y + d)(z + d)
\end{aligned}
$$

在上式中，视$x,y,z$为常数，比较两边$d^4$的系数，容易发现$k = 1$，再把$x,y,z$带回$a,b,c$，综上，有因式分解结果

$$
\begin{aligned}
 &(a+b+c+d)(b+c-a-d)(c+a-b-d)(a+b-c-d) + 16 abcd \\
 = & (a+b+c-d)(b+c-a+d)(c+a-b+d)(a+b-c+d)
\end{aligned}
$$

### 15

已知正实数 $a, b, c$ 满足 $abc = 1$，证明：

$$
a(a + 1) + b(b + 1) + c(c + 1) \geqslant 6
$$

***Solution***

将原式拆分为平方和与一次项和，分别利用 **AM-GM 均值不等式**证明即可。这里整理了三种证明方法，其中**直接使用六元 AM-GM** 的方法最为巧妙，另外**切线法**也很有启发性。

---

#### 方法一：均值不等式拆分法（达叔的方法）

将待证不等式展开：
$$
a(a+1) + b(b+1) + c(c+1) = (a^2 + b^2 + c^2) + (a + b + c)
$$

因为 $a, b, c > 0$ 且 $abc = 1$：
1. 由三元 **AM-GM 均值不等式**有：
   $$
   a + b + c \geqslant 3 \sqrt[3]{abc} = 3 \sqrt[3]{1} = 3
   $$
   当且仅当 $a = b = c = 1$ 时等号成立。
2. 同理，对平方项使用三元 **AM-GM 均值不等式**：
   $$
   a^2 + b^2 + c^2 \geqslant 3 \sqrt[3]{a^2 b^2 c^2} = 3 \sqrt[3]{(abc)^2} = 3
   $$
   当且仅当 $a = b = c = 1$ 时等号成立。

两式相加，即得：
$$
(a^2 + b^2 + c^2) + (a + b + c) \geqslant 3 + 3 = 6
$$
即 $a(a + 1) + b(b + 1) + c(c + 1) \geqslant 6$，等号在 $a = b = c = 1$ 时成立。

---

#### 方法二：六元均值不等式法（最简捷）

我们直接把展开后的六个正数项 $a^2, b^2, c^2, a, b, c$ 作为一个整体，直接应用**六元 AM-GM 均值不等式**：
$$
\frac{a^2 + b^2 + c^2 + a + b + c}{6} \geqslant \sqrt[6]{a^2 \cdot b^2 \cdot c^2 \cdot a \cdot b \cdot c} = \sqrt[6]{(abc)^3} = \sqrt{abc} = 1
$$

两边同乘以 $6$，立刻得到：
$$
a^2 + b^2 + c^2 + a + b + c \geqslant 6
$$
由于所有项都为正数，等号成立条件为：
$$
a^2 = b^2 = c^2 = a = b = c
$$
结合 $a, b, c > 0$，可得 $a = b = c = 1$。这只需一步即可完成证明，非常漂亮！

---

#### 方法三：切线法（局部不等式法）

设函数 $f(x) = x(x+1) = x^2 + x$。我们要证明 $\sum f(a) \geqslant 6$。
注意到等号在 $a=b=c=1$ 时取得，我们求 $f(x)$ 在 $x=1$ 处的切线方程。
- $f(1) = 2$
- $f'(x) = 2x + 1 \implies f'(1) = 3$

因此，切线方程为 $y - 2 = 3(x - 1) \implies y = 3x - 1$。
我们来验证局部不等式 $f(x) \geqslant 3x - 1$ 是否对所有正实数成立：
$$
f(x) - (3x - 1) = x^2 + x - 3x + 1 = x^2 - 2x + 1 = (x-1)^2 \geqslant 0
$$
该不等式显然对所有实数 $x$ 都成立。

因此，我们有以下局部不等式：
$$
\begin{aligned}
a(a+1) &\geqslant 3a - 1 \
b(b+1) &\geqslant 3b - 1 \
c(c+1) &\geqslant 3c - 1
\end{aligned}
$$
将三式相加：
$$
a(a+1) + b(b+1) + c(c+1) \geqslant 3(a+b+c) - 3
$$
再由三元 **AM-GM 不等式**，$a+b+c \geqslant 3 \sqrt[3]{abc} = 3$，代入上式得：
$$
a(a+1) + b(b+1) + c(c+1) \geqslant 3(3) - 3 = 6
$$
等号在 $a = b = c = 1$ 时成立。
