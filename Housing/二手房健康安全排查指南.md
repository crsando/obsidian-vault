# 二手房健康安全排查指南

为了保护孕妇、老人和婴儿，购买或翻新二手房（特别是10-15年房龄）时，必须进行以下健康与安全检测：

## 1. 室内空气“四大杀手”检测
- **重点防范：** 胎儿畸变、婴幼儿白血病。
- **检测项目：** 甲醛、苯系物、TVOC。
- **特殊关注：** **氡气**。如果房屋使用了大量天然大理石或花岗岩（如飘窗、背景墙、厨房台面），必须加测氡气（一种有放射性、易致肺癌的气体）。
- **建议方案：** 寻找带 **CMA资质** 的专业机构上门采样化验，不要依赖便携式简易仪器。

## 2. 管道水质检测
- **重点防范：** 重金属超标。老小区的入户水管容易老化。
- **检测项目：** 铅、铁等重金属，以及大肠杆菌。
- **建议方案：** 
  - 顺便让CMA机构取水样化验。
  - **保底措施：** 在水井或入户处加装“前置过滤器”，厨房水槽下安装“RO反渗透净水器”。

## 3. 用电与燃气安全检测
- **重点防范：** 意外触电或燃气泄漏，保护儿童安全。
- **检测项目：**
  - **电路：** 请电工师傅检测全屋插座，确认 **接地线** 是否正常（老房子地线常有缺失），测试配电箱 **漏电保护器** 的灵敏度。
  - **燃气：** 请燃气公司上门检测老旧管道接口是否存在微漏。

## 4. 霉菌与物理安全排查
- **防霉菌：** 重点检查卫生间相邻墙壁、墙纸接缝、实木地板发黑处。若有发霉，说明存在漏水暗疾或通风极差（如全南户型不通风）。
- **防坠落：** 检查飘窗和南向大阳台的护栏是否牢固，窗户是否能加装 **儿童安全锁**。

## 附：评估房间霉菌与细菌的简易方法
- **自然沉降法：** 网上购买营养琼脂培养皿，在主卧、卫生间等处静置5-15分钟，观察48小时后的菌落数量（白色通常是细菌，彩色毛茸茸通常是霉菌）。适合用来对比新风开启前后的效果。
