## 当前状况

- **现住房**：张江核心区（中环内），85㎡两室一厅，2012年建成
- **预估卖出价**：**550万**
- **家庭计划**：计划一两年内生孩子，需考虑老人帮带、居家办公空间
- **房价判断**：短期可能反弹，**长期不看好**
- **购房方式**：全款，不贷款

## 三套候选方案

| 维度 | 方案① 张江核心老房 | 方案② 张江外围次新 | 方案③ 不换房+同小区租一套 |
|------|-------------------|-------------------|------------------------|
| 面积 | 117㎡ | ~110㎡ | 85㎡ + 租房 |
| 房龄 | 2005年（21年） | 2018年（8年） | 自住2012年（14年） |
| 买入价 | **750-800万** | **800-850万** | 不买 |
| 小学学区 | ✅ 确定OK | ⚠️ 不确定，有风险 | 维持现状 |
| 停车 | ⚠️ 老小区，可能不便 | ✅ 地库标配 | 不变 |

## 财务测算

### 基础假设

| 参数 | 数值 |
|------|------|
| 现房卖出价 | 550万 |
| 稳健理财年化收益 | 3.5% |
| 老房（2005年）年折旧率 | 2% |
| 次新房（2018年）年折旧率 | 1.5% |
| 现住房（2012年）年折旧率 | 1.5% |
| 方案③月租金 | 8,500元 |

### 方案① 全款换张江核心老房（750-800万）

**一次性资金占用：**

| 项目 | 金额 |
|------|------|
| 差价（买入 - 卖出） | 200-250万 |
| 交易税费 + 中介费（买卖双边） | 30-40万 |
| 装修（2005年老房需大改） | 25-35万 |
| **合计额外资金占用** | **255-325万** |

**年度隐性成本（vs 不动）：**

| 项目 | 金额/年 |
|------|---------|
| 资金机会成本（255-325万 × 3.5%） | 8.9-11.4万 |
| 新增折旧风险（750-800万×2% − 550万×1.5%） | 6.75-7.75万 |
| **年度额外隐性成本** | **≈15.7-19.2万** |

> ⚠️ 2005年房子到孩子上小学时房龄将达 **27-28年**，届时出手流动性堪忧。

### 方案② 全款换张江外围次新房（800-850万）

**一次性资金占用：**

| 项目 | 金额 |
|------|------|
| 差价（买入 - 卖出） | 250-300万 |
| 交易税费 + 中介费 | 35-45万 |
| 装修（次新房轻装即可） | 10-15万 |
| **合计额外资金占用** | **295-360万** |

**年度隐性成本（vs 不动）：**

| 项目 | 金额/年 |
|------|---------|
| 资金机会成本（295-360万 × 3.5%） | 10.3-12.6万 |
| 新增折旧风险（800-850万×1.5% − 550万×1.5%） | 3.75-4.5万 |
| **年度额外隐性成本** | **≈14.1-17.1万** |

> 次新房折旧更慢，年度隐性成本略低于方案①，但**资金占用最大，手头现金最紧张**。

### 方案③ 不换房 + 同小区租一套（月租8,500）

**年度成本：**

| 项目 | 金额/年 |
|------|---------|
| 租金支出（8,500 × 12） | 10.2万 |
| 交易成本 | 0 |
| 额外资金占用 | 0 |
| **年度额外成本** | **≈10.2万** |

> 省下来的 255-360万 做 3.5% 理财，年收益 **9-12.6万**，几乎完全覆盖租金。
> 扣除理财收益后，**实际年净成本仅 0-1.2万**。

## 综合对比

| | 方案① 老房 | 方案② 次新 | 方案③ 不换+租 |
|--|-----------|-----------|-------------|
| 一次性资金占用 | 255-325万 | 295-360万 | **0** |
| 年度额外隐性成本 | **15.7-19.2万** | **14.1-17.1万** | **10.2万** |
| 理财收益抵扣后净成本 | 15.7-19.2万 | 14.1-17.1万 | **0-1.2万** |
| 手头现金 | 紧张 | 很紧张 | **充裕** |
| 可逆性 | 低 | 低 | **高（随时退租）** |
| 居住体验 | 宽敞但硬件老化 | 最好 | 分两套，灵活但不整 |
| 学区确定性 | ✅ 确定 | ⚠️ 有风险 | 维持现状 |
| 资产折旧风险 | ⛔ 最高 | 中等 | ✅ 最低 |

## 核心结论

1. **在看空房价 + 全款买入的前提下，方案③的财务优势是碾压级的。** 年度净成本几乎为零，同时保留了最大的灵活性和现金储备。

2. **方案①财务上最不划算。** 资金锁死在加速贬值的老房子里，年度隐性成本是方案③的十倍以上。唯一优势是学区确定。

3. **方案②逻辑最自洽（自住消费品思路），但资金占用最大。** 如果能确认对口小学至少中等，且经济上能承受手头现金吃紧，可以考虑。

4. **方案③本质上是花极低成本买了几年的决策缓冲期。** 等孩子出生后需求更清晰、房价走势更明朗再做决定，决策质量会显著高于现在。

5. **唯一能推翻方案③优势的场景：房价大幅反弹。** 但这恰恰是当前不看好的方向。

## 方案③实操要点

- 同小区租一套做**居家办公 + 老人住房**，物理隔离工作与家庭
- 与家人明确"上班时间"边界，租房不是逃避带娃
- 省下来的资金做稳健理财（3-4%），收益覆盖租金
- 等孩子 3-4 岁、需求清晰时再评估是否换房
- 届时对口小学政策、房价走势、家庭需求都更明确
