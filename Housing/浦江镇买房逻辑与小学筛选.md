## 核心结论

浦江镇 / 浦锦板块不适合按“强学区房”逻辑买，更适合按：

> **居住品质 + 次新/准次新商品房 + 110㎡以上改善户型 + 公办小学不差 + 总价别溢价太狠**

当前筛选条件：

1. **面积 110㎡以上**
2. **小区年份 2000-2017 年之间**，不看新房
3. **对口公办小学还可以**，优先考虑：
   - **上海师范大学附属闵行第三小学，主校**
   - **黄浦一中心世博小学**
   - **上海师范大学附属闵行第三小学南校，原浦江第一小学**
4. 买房本质仍然是改善，不是赌学区升值。

## 重要辨析：几个学校不要混

### 1. 上海师范大学附属闵行第三小学，主校

- 地址：**南江榉路591号**
- 官方划片中对应的是“上海师范大学附属闵行第三小学”
- 这是浦江 / 浦锦片区里更值得优先看的公办小学
- 对口小区整体商品房和改善属性更强

### 2. 上海师范大学附属闵行第三小学南校

- 地址：**立跃路3939号**
- 即原来的 **闵行区浦江第一小学**
- 2022 年正式更名为上海师范大学附属闵行第三小学南校
- 是百年老校，有历史积累，但和南江榉路主校不是同一个划片

### 3. 上海市闵行区浦江第三小学

- 地址：**鲁浩路350号**
- 对口鲁汇、瑞和城等片区
- 不建议作为“三梯队以上小学”核心买房筛选目标

### 4. 黄浦一中心世博小学

- 地址：**浦驰路177号**
- 2006 年创建
- 学校管理团队来自黄浦区优质教育资源，有一定品牌背书
- 学校本身不错，但对口小区中世博家园占比很高，小区属性和居住品质需要严查

## 第一优先：上师大附属闵行第三小学主校对口

官方 2025 划片中，主校对口范围包括：

| 小区 | 地址/弄号 | 初步判断 |
|---|---|---|
| **浦江华侨城** | 江桦路200弄、400弄、600弄、199弄、399弄；江柳路200弄、400弄；浦驰路1809弄、1659弄、1335弄、1336弄、1628弄；浦锦路1658弄；江栀路500弄、350弄；浦申路1288弄；浦晓路112弄 | 板块标杆，品质好，但总价和溢价较高 |
| **万科翡翠别墅** | 江桃路555弄、556弄；浦秀北路55弄、155弄 | 别墅/高端属性，不符合普通改善筛选 |
| **红醍半岛（涵云雅庭）** | 江柳路888弄、889弄 | 偏高端，总价高 |
| **保利茉莉公馆** | 浦秀路1535弄、1536弄；北江桐路536弄 | 重点可看，品牌、年份、改善属性较好 |
| **浦江坤庭** | 浦秀路1385弄、1386弄；浦雪路1180弄；浦锦路1181弄 | 重点可看，综合匹配度高 |
| **浦江颐城** | 浦泉路251弄、399弄；浦晓路298弄 | 重点可看，可能有性价比 |
| 桂鸿院 | 江柳路600号 | 需单独核实 |
| 桃花源墅 | 江桃路399弄、299弄、199弄 | 别墅属性，不作为主筛选 |
| 一品漫城 | 浦星公路568弄、芦恒路378弄、浦申路2639弄 | 官方标注为“暂时安置”，要谨慎 |

### 这一组的买房逻辑

这是当前浦江/浦锦片区里最值得优先看的组合。

理由：

- 学校比普通浦江镇小学更有想象力
- 小区商品房、改善属性更明确
- 110㎡以上户型更容易匹配改善需求
- 小区年份大多可能落在 2000-2017 区间或接近该区间
- 相比真正顶级学区，价格还没有完全按强学区逻辑定价

### 推荐排序

#### 1. **浦江坤庭**

优先级最高之一。

优点：

- 对口 **上师大附属闵行第三小学主校**
- 小区改善属性比较强
- 110㎡以上户型更容易匹配
- 比浦江华侨城可能更现实

注意：

- 不要为“上师大附属”四个字追高
- 要实地核停车、楼栋位置、户型采光
- 如果价格接近张江/御桥优质小区，需要重新比较通勤和板块价值

#### 2. **保利茉莉公馆**

优先级很高。

优点：

- 对口主校
- 保利品牌
- 公开资料显示部分房源/小区建成约 2012 年，符合年份要求
- 改善属性较好

注意：

- 小区不同期、不同楼栋要分开看
- 重点核实 110㎡以上户型的总价、停车、楼栋噪音

#### 3. **浦江颐城**

作为性价比候选。

优点：

- 对口主校
- 改善属性较好
- 可能比华侨城更容易控制总价

注意：

- 要核年份、物业、楼栋品质
- 如果户型或楼栋位置一般，不能只因学校买

#### 4. **浦江华侨城**

板块标杆，但不一定最适合。

优点：

- 小区品质和板块形象好
- 对口主校
- 保值性可能更好

缺点：

- 总价压力大
- 容易叠加“品质 + 板块 + 学校”复合溢价
- 如果预算卡得紧，性价比未必好

| **世博家园** | 浦驰路345弄、188弄、235弄 |
| **世博家园** | 浦晓南路59弄 |

| **智汇园** | 联航路1399弄 |
| **满香苑** | 江龙路446弄 |

### 这一组的买房逻辑

学校本身不错，可以作为“公办小学还可以”的筛选条件。  
但对口小区中，**世博家园**占比很高，而世博家园很多街坊存在动迁/保障/商品混合属性，居住品质和改善属性需要逐个街坊看。

### 世博家园

优点：

- 对口黄浦一中心世博小学
- 小区年代大多在 2005-2006 左右，符合年份要求
- 生活配套成熟
- 总价相对可控
- 大面积户型可能能找到

缺点：

- 小区体量大，不符合“小而美”偏好
- 动迁/商品混合属性要核实
- 物业、停车、楼道维护、居民结构差异大
- 学校加分，但小区本身可能扣分

判断：

> **可以作为性价比备选，但不是改善首选。**

### 智汇园 / 满香苑

这两个需要进一步核：

- 是否有 110㎡以上户型
- 小区年份
- 商品房属性
- 交易流动性
- 停车情况
- 是否存在明显噪音或配套短板

如果小区品质明显优于普通世博家园街坊，可以优先于世博家园看。

## 实地看房核查清单

### 小区层面

- 是否纯商品房，还是动迁/商品混合
- 建成年份是否在 **2000-2017 年**
- 物业费和物业维护水平
- 地下车库是否充足
- 是否能买车位，或长期稳定租车位
- 小区是否过大、是否嘈杂
- 楼栋是否靠近主干道、高架、商业、垃圾站、变电站
- 110㎡以上户型是否南北通透
- 是否有两卫
- 得房率和房型是否适合改造

### 学校层面

- 以当年官方划片为准，不听中介口述
- 注意区分：
  - **上师大附属闵行第三小学主校**
  - **上师大附属闵行第三小学南校，原浦江第一小学**
  - **浦江第三小学**
- 如果小区属于“暂时安置”，要格外谨慎
- 不要为了小学支付过高溢价

### 价格层面

- 如果总价接近张江/御桥目标小区，应重新比较：
  - 通勤
  - 停车
  - 小区品质
  - 板块保值性
  - 未来置换流动性
- 浦江/浦锦的优势应该是“面积和居住感”，不是“强学区”。

---

## 一句话策略

> **优先看上师大附属闵行第三小学主校对口的浦江坤庭、保利茉莉公馆、浦江颐城；预算充足再看浦江华侨城；世博小学对口小区作为性价比备选；原浦江一小/南校只作为学校不拖后腿的加分项，不为它单独支付高溢价。**
