# 暑期几何专题课程：全等三角形（8讲）

## 课程定位

- **适用对象**：上海沪教版体系，初一升初二理科班学生（沪教版七年级下册第14章衔接拔高）
- **课程目标**：系统掌握全等判定 → 熟练经典模型 → 突破辅助线构造 → 衔接竞赛拓展
- **每讲时长建议**：120分钟（讲授60min + 练习与讲评60min）

---

## 第1讲：全等三角形的概念与五大判定方法

| 项目 | 内容 |
|------|------|
| **核心知识** | 全等形/全等三角形概念、对应边与对应角的寻找方法 |
| **五大判定** | SSS、SAS、ASA、AAS、HL（直角三角形专用） |
| **重点拓展** | 为什么 SSA 不能判定全等？（反例构造与几何直观） |
| **理科班加餐** | SSA 在特殊条件下成立的讨论（如大边对大角约束下） |
| **课堂目标** | 熟记判定条件，建立凑条件的基本意识 |

---

## 第2讲：全等证明的基本思路与规范书写

| 项目 | 内容 |
|------|------|
| **方法论** | 证明全等的三步走：①确定目标三角形对 → ②盘点已知条件 → ③选择判定方法 |
| **思路训练** | 边角边路线 vs 角边角路线的选择策略 |
| **典型应用** | 用全等证明线段相等、角度相等、直线平行/垂直 |
| **书写规范** | 严格的∵…∴…证明格式（上海中考评分要求） |
| **理科班加餐** | 一题多解训练——同一道题用不同判定方法证明 |

---

## 第3讲：角平分线模型

| 项目 | 内容 |
|------|------|
| **核心模型** | 角平分线的性质定理与判定定理 |
| **三种构造法** | ①向两边作垂线构造全等 ②作平行线构造等腰三角形 ③在角两边截等长线段构造全等 |
| **经典题型** | 角平分线 + 平行线组合；角平分线 + 垂直组合 |
| **理科班加餐** | 三角形三条角平分线交于一点（内心）的证明思路引导 |
| **课堂目标** | 看到角平分线条件能条件反射地联想辅助线方案 |

---

## 第4讲：旋转型全等——手拉手模型与半角旋转模型

| 项目 | 内容 |
|------|------|
| **手拉手模型** | 两个三角形共顶点、顶角相等，绕公共顶点旋转后重合 |
| **三个层次** | ①等边三角形手拉手 → ②等腰直角三角形手拉手 → ③一般等腰三角形手拉手 |
| **半角旋转模型** | 旋转角为某角一半时的全等构造 |
| **识别要诀** | 公共顶点
