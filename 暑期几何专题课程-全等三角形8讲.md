# 暑期几何专题课程：全等三角形（8讲）

## 课程定位

- **适用对象**：上海沪教版体系，初一升初二理科班学生（沪教版七年级下册第14章衔接拔高）
- **课程目标**：系统掌握全等判定 → 熟练经典模型 → 突破辅助线构造 → 衔接竞赛拓展
- **每讲时长建议**：120分钟（讲授60min + 练习与讲评60min）
- **课程关键词**：全等判定、证明规范、角平分线、手拉手模型、倍长中线、截长补短、二次全等

---

## 一、设计依据与课程思路

### 1. 上海教材与考查特点

上海沪教版初中数学中，全等三角形一般出现在七年级下册《三角形》相关章节，是学生从“计算型几何”过渡到“证明型几何”的重要分水岭。

这个板块不仅要求学生会判断两个三角形全等，更重要的是：

1. 能准确寻找对应边、对应角；
2. 能根据条件选择合适的判定方法；
3. 能用全等证明线段相等、角相等、平行、垂直等结论；
4. 能通过辅助线主动构造全等三角形；
5. 能在复杂图形中识别经典几何模型。

### 2. 理科班学生的课程定位

理科班学生通常基础较好，不能只停留在“SSS、SAS、ASA、AAS、HL”的机械训练上。课程应当突出三点：

- **模型化**：把零散题目归纳成固定模型，减少盲目刷题。
- **方法化**：训练“缺什么补什么”的辅助线意识。
- **综合化**：从单次全等走向二次全等、多模型融合。

因此，这8讲建议采用：

> 基础判定 → 证明规范 → 经典模型 → 辅助线构造 → 综合提高

这样的递进结构。

# 第1讲：全等三角形的概念与五大判定方法

## 课程目标

让学生从直观图形进入严格证明，掌握全等三角形的基本概念、性质和五种常用判定方法。

## 核心知识

### 1. 全等三角形的概念

两个三角形能够完全重合，则称这两个三角形全等。

全等后：

- 对应边相等；
- 对应角相等；
- 周长相等；
- 面积相等。

### 2. 对应关系的寻找

学生最容易错的是“对应点写错”。建议强调：

- 看角度大小对应；
- 看边长对应；
- 看公共边、公共角；
- 看字母顺序。

例如：

若 △ABC ≌ △DEF，则对应关系是：

- A ↔ D
- B ↔ E
- C ↔ F
- AB ↔ DE
- BC ↔ EF
- AC ↔ DF

### 3. 五大判定方法

| 判定 | 中文名称 | 使用条件 |
|---|---|---|
| SSS | 边边边 | 三边对应相等 |
| SAS | 边角边 | 两边及夹角对应相等 |
| ASA | 角边角 | 两角及夹边对应相等 |
| AAS | 角角边 | 两角及一边对应相等 |
| HL | 斜边直角边 | 直角三角形中，斜边和一条直角边对应相等 |

## 重点拓展：为什么 SSA 不能判定全等？

这是理科班必须讲清楚的点。

SSA 指的是“两边和其中一边的对角对应相等”。一般情况下，SSA 不能判定两个三角形全等，因为可能出现一边固定、一个角固定，但第三个点有两个不同位置的情况，形成两个不全等三角形。

但可以进一步讨论：

- 如果是直角三角形，则转化为 HL；
- 如果给定的是“大边对大角”或附加唯一性条件，某些特殊情形下可以确定三角形；
- 但初中证明中不能直接写“SSA”。

## 课堂训练重点

1. 判断两个三角形是否全等；
2. 写出对应边、对应角；
3. 根据已知条件选择判定方法；
4. 识别 SSA 陷阱。

## 理科班加餐

设计一组“看起来像全等但其实不一定全等”的反例题，让学生体会判定定理的边界。

# 第2讲：全等证明的基本思路与规范书写

## 课程目标

把学生从“知道判定”推进到“能写证明”。重点解决两个问题：

1. 不知道证哪两个三角形全等；
2. 证明过程书写不规范。

## 方法论：全等证明三步走

### 第一步：找目标三角形

从结论倒推。

- 要证明线段相等，找包含这两条线段的两个三角形；
- 要证明角相等，找包含这两个角的两个三角形；
- 要证明中点、平行、垂直，也常常先转化为边或角相等。

### 第二步：盘点已知条件

常见条件来源：

- 题目直接给出；
- 公共边；
- 公共角；
- 对顶角相等；
- 平行线产生的内错角、同位角；
- 垂直产生的直角；
- 中点产生的线段相等；
- 角平分线产生的角相等。

### 第三步：选择判定方法

根据已有条件判断走哪条路线：

- 边多：考虑 SSS 或 SAS；
- 角多：考虑 ASA 或 AAS；
- 有直角：优先考虑 HL；
- 缺条件：考虑添加辅助线。

## 证明书写规范

建议统一格式：

```text
在 △ABC 和 △DEF 中，
∵ AB = DE，
   ∠ABC = ∠DEF，
   BC = EF，
∴ △ABC ≌ △DEF（SAS），
∴ AC = DF，∠A = ∠D。
```

强调：

- 三角形字母顺序必须对应；
- 判定理由必须写清楚；
- 由全等推出结论时，要写“对应边/对应角相等”。

## 典型应用

1. 用全等证明线段相等；
2. 用全等证明角相等；
3. 用全等证明两线平行；
4. 用全等证明两线垂直；
5. 用全等证明点是中点。

## 理科班加餐

同一道题要求学生至少写出两种证明路线，比如：

- 路线一：SAS；
- 路线二：ASA；
- 路线三：先证角相等，再证 AAS。

目的不是炫技，而是训练学生的“条件调度能力”。

# 第3讲：角平分线模型

## 课程目标

掌握角平分线相关的全等构造。角平分线是全等中最常见的“造角相等”条件，也是辅助线训练的好入口。

## 核心模型

### 1. 角平分线性质定理

角平分线上的点到角两边的距离相等。

证明通常通过构造两个直角三角形全等，使用 HL 或 AAS。

### 2. 角平分线判定定理

角内部一点到角两边距离相等，则该点在角平分线上。

## 三种常见构造法

### 方法一：向角两边作垂线

适用场景：题目中有“距离”“垂直”“到两边”等信息。

构造：

- 从角平分线上一点分别向角两边作垂线；
- 得到两个直角三角形；
- 用 HL 或 AAS 证明全等。

### 方法二：作平行线，构造等腰三角形

适用场景：角平分线与平行线同时出现。

常见结论：

- 角平分线 + 平行线 → 等腰三角形；
- 等腰三角形再提供边相等，为全等创造条件。

### 方法三：在角的两边截取相等线段

适用场景：需要人为制造两条相等边。

构造：

- 在角的两边分别截取相等长度；
- 利用角平分线提供夹角相等；
- 通过 SAS 构造全等。

## 经典题型

1. 角平分线 + 垂直；
2. 角平分线 + 平行线；
3. 角平分线 + 中点；
4. 角平分线 + 线段和差；
5. 角平分线 + 截长补短。

## 理科班加餐

引导证明：三角形三条角平分线交于一点。

思路：

- 两条角平分线交于一点 P；
- P 到两边距离相等；
- 再证明 P 到第三边距离也相等；
- 从而 P 在第三条角平分线上。

这个内容可以作为内心概念的预备。

# 第4讲：旋转型全等——手拉手模型与半角旋转模型

## 课程目标

让学生掌握全等三角形中最重要的旋转型模型。很多竞赛题、中考压轴题，本质都是“旋转后构造全等”。

## 一、手拉手模型

### 1. 模型特征

手拉手模型通常指：

- 两个三角形有公共顶点；
- 公共顶点处的两个角相等或互补；
- 两组对应边相等；
- 图形可以看成绕公共顶点旋转得到。

直观记忆：

> 公共顶点伸出四条线段，像两个人手拉手。

### 2. 常见类型

#### 类型一：等边三角形手拉手

常见条件：

- △ABC 和 △ADE 都是等边三角形；
- 可得 AB = AC，AD = AE，∠BAC = ∠DAE = 60°；
- 常证 △BAD ≌ △CAE。

#### 类型二：等腰直角三角形手拉手

常见条件：

- 两个等腰直角三角形共顶点；
- 利用 90° 旋转构造 SAS；
- 常用于正方形、直角三角形相关题。

#### 类型三：一般等腰三角形手拉手

常见条件：

- 两个等腰三角形顶角相等；
- 两组腰分别相等；
- 通过 SAS 证明旋转后的三角形全等。

## 二、半角旋转模型

### 1. 模型特征

半角旋转模型常见于：

- 一个角被分成两半；
- 图形中存在等边、等腰或正方形结构；
- 需要旋转某条线段，使两个分散条件集中到同一个三角形中。

### 2. 解题关键

- 找旋转中心；
- 找旋转角；
- 旋转一条边或一个三角形；
- 证明旋转前后构成全等。

## 识别口诀

> 共顶点、等边长、等角度，多半就是旋转全等。

## 理科班加餐

正方形中的手拉手模型：

- 正方形天然提供四条等边、四个直角；
- 常与等腰直角三角形组合；
- 可训练学生识别 90° 旋转。

# 第5讲：对称型全等——背对背模型与一线三等角/三垂直

## 课程目标

训练学生识别“看起来不在一起，但其实可以配对”的三角形结构。重点是共边、共线、垂直、等角带来的全等机会。

## 一、背对背模型

### 1. 模型特征

两个三角形：

- 共用一条边；
- 分别位于公共边两侧；
- 通过边相等、角相等或轴对称关系证明全等。

### 2. 常见应用

- 证明公共边两侧的线段相等；
- 证明某条线是角平分线或垂直平分线；
- 与等腰三角形、垂直平分线结合。

## 二、一线三垂直模型

### 1. 模型特征

“一线三垂直”通常指出现：

- 三个点在同一条直线上；
- 有两条或三条垂线；
- 形成两个直角三角形；
- 利用直角、余角关系、线段相等构造全等。

### 2. 常见结论

- 线段相等；
- 角相等；
- 面积关系；
- 直角三角形全等。

## 三、一线三等角模型

### 1. 模型特征

- 多个点共线；
- 出现三个相等角；
- 通过外角、邻补角、平行线角关系制造 ASA/AAS。

### 2. 解题关键

- 不只看图中标出的角；
- 要主动利用共线产生的 180°；
- 用等角的补角也相等。

## 四、等腰三角形中的边边角模型

一般 SSA 不能判定全等，但在某些等腰结构中，可以通过“等边对等角”补出角条件，使其转化为 ASA、AAS 或 SAS。

## 理科班加餐

给出多幅图，让学生只判断模型类型，不急着计算或证明。

训练目标：

> 先识图，再证明。

# 第6讲：辅助线专题（一）——倍长中线法

## 课程目标

掌握全等中最经典的辅助线之一：倍长中线法。它是处理“中点、中线、线段关系”的核心工具。

## 一、适用场景

看到以下条件要敏感：

- 中点；
- 中线；
- 某点是某线段中点；
- 线段被平分；
- 结论涉及两条边的和、差或不等关系。

## 二、核心操作

若 D 是 BC 的中点，AD 是中线，则常作：

> 延长 AD 至 E，使 DE = AD。

于是：

- AD = DE；
- BD = DC；
- ∠BDA = ∠CDE（对顶角）；
- 可证 △ABD ≌ △ECD（SAS）。

这一步的作用是：

> 把原来分散在三角形两侧的条件“搬运”到一个新的三角形里。

## 三、经典题型

### 题型一：中线 + 线段相等

通过倍长中线得到新的全等三角形，再转化边相等。

### 题型二：中线 + 角度关系

倍长后产生平行、等角或等腰结构。

### 题型三：中线 + 三角形三边关系

倍长中线后，把问题转化为三角形两边之和大于第三边。

### 题型四：中点 + 平行线

中点和平行结合，常可构造中位线或全等。

## 四、理科班加餐

### 1. 倍长非中线

有些题虽然没有直接给“中线”，但可以通过构造一个相等线段，使其变成类似倍长中线结构。

### 2. 逆向倍长

从结论倒推：

- 如果结论中出现两条线段之和；
- 或出现一条线段与另一条线段的倍数关系；
- 可以考虑延长其中一条线段。

# 第7讲：辅助线专题（二）——截长补短法

## 课程目标

掌握处理“线段和差关系”的核心辅助线方法。截长补短是全等证明中最能体现技巧的板块之一。

## 一、适用场景

当题目要求证明：

- AB = AC + CD；
- AB + AC = AD；
- AB - AC = CD；
- 某条线段等于另外两条线段的和或差；

就要优先想到截长补短。

## 二、截长法

### 1. 操作方式

在较长线段上截取一段，使其等于某条已知短线段。

例如：

若要证明 AB = AC + CD，可在 AB 上取点 E，使 AE = AC，再证明 EB = CD。

### 2. 思维本质

把“整体等于部分之和”拆成两个更容易证明的线段相等。

## 三、补短法

### 1. 操作方式

把较短线段延长，使延长后的线段等于另一条线段。

例如：

要证明 AB + AC = AD，可延长 AB 到 E，使 BE = AC，再证明 AE = AD。

### 2. 思维本质

把“加法关系”变成“两个整段相等”。

## 四、经典组合：截长补短 + 角平分线

这是中考和竞赛里非常高频的组合。

常见结构：

- 已知 AD 平分 ∠A；
- 结论涉及 AB、AC、BD、CD 的和差；
- 在角的两边截取相等线段；
- 利用 SAS 构造全等。

## 五、理科班加餐

### 1. 多次截补

复杂题中可能需要连续截取两次或先补后截。

### 2. 与外角定理结合

截长补短后，新产生的三角形可能需要利用外角关系证明角相等。

### 3. 与旋转模型结合

有些题表面是线段和差，实质是旋转后线段拼接。

# 第8讲：综合提高与二次全等

## 课程目标

把前7讲的知识整合起来，训练学生处理复杂几何证明题的能力。

## 一、什么是二次全等

二次全等指：

1. 先证明第一组三角形全等；
2. 得到一批中间结论，比如边相等、角相等；
3. 再利用这些中间结论证明第二组三角形全等；
4. 最终推出题目要求的结论。

## 二、二次全等的常见信号

- 题目结论离已知条件较远；
- 一次全等后只能得到部分条件；
- 图中有三到四个三角形可以配对；
- 需要先证一个“桥梁结论”。

## 三、综合模型融合

### 1. 手拉手 + 截长补短

旋转构造全等后，常常得到线段拼接关系。

### 2. 角平分线 + 倍长中线

角平分线提供角相等，中线提供边相等，二者可共同服务于 SAS 或 AAS。

### 3. 背对背 + 一线三垂直

共边结构与直角结构结合，常出现 HL 或 AAS。

### 4. 正方形 + 旋转全等

正方形天然提供：

- 四边相等；
- 四个直角；
- 对角线性质；
- 90°旋转结构。

## 四、课堂安排建议

第8讲不建议继续灌输大量新模型，而应安排：

1. 模型识别训练；
2. 综合例题精讲；
3. 学生独立思考；
4. 多解法展示；
5. 课后错题归类。

## 五、理科班加餐

精选上海中考压轴题、希望杯、中环杯或同等级几何题，要求学生完成：

- 标出目标三角形；
- 写出已有条件；
- 判断缺失条件；
- 选择辅助线；
- 完整书写证明。

# 五、题目难度配比建议

每讲建议配置：

- **课堂例题**：6-8道；
- **课后练习**：10道；
- **难度比例**：基础 : 提高 : 竞赛 = 3 : 4 : 3。

## 基础题

用于巩固判定和书写，例如：

- 直接证明两个三角形全等；
- 利用全等求角度或线段；
- 判断对应关系。

## 提高题

用于训练模型识别，例如：

- 角平分线模型；
- 手拉手模型；
- 背对背模型；
- 倍长中线；
- 截长补短。

## 竞赛题

用于训练综合能力，例如：

- 二次全等；
- 多模型融合；
- 需要主动添加辅助线；
- 一题多解。

# 七、教学注意事项

## 1. 不要一开始就堆模型

如果学生证明规范不扎实，模型越多越乱。前两讲必须把“全等证明的基本语言”打牢。

## 2. 模型要配图讲

全等模型高度依赖图形直觉，建议每个模型都配标准图、变式图、隐藏图三类。

## 3. 每讲都要训练“缺什么补什么”

几何证明不是背答案，而是看当前判定缺什么条件，再想办法制造这个条件。

## 4. 理科班要讲边界条件

比如 SSA 为什么不行、什么时候特殊成立；这类问题能明显提升学生的数学严谨性。

## 5. 第8讲不要变成纯刷题

综合课应当以“归类、复盘、迁移”为主，而不是单纯讲难题。
