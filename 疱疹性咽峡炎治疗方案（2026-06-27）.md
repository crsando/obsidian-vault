## 诊断：疱疹性咽峡炎（病毒性溃疡性咽炎）

**体重：** 110kg  
**病程预估：** 溃疡痛感还有2-3天高峰期，之后逐渐减轻，总计约5-7天愈合。

## 二、当前使用药物

### 1. 布洛芬混悬液（20mg/ml）—— 全身止痛

| 项目 | 内容 |
|------|------|
| 单次剂量 | **400-600mg（20-30ml）** |
| 频率 | 每 **6-8小时** 一次 |
| 每日上限 | 不超过 **1200mg（60ml）**，即最多3次 |
| 服用时间 | 饭前30分钟（止痛后好进食）+ 睡前 |
| 注意 | 胃不舒服改饭后；夜间痛重可用600mg（30ml） |

### 2. 复方氯己定含漱液 —— 消毒防感染

| 项目 | 内容 |
|------|------|
| 用法 | 含漱30秒后吐掉 |
| 频率 | 每天 **2-3次**，饭后用 |
| 作用 | 保持溃疡面清洁，防止继发细菌感染 |
| 注意 | 不要超过一天4次，长期高频用会使牙齿/舌头染色；含漱后30分钟内不吃不喝 |

### 3. 康复新液 —— 促溃疡愈合（建议购买）

| 项目 | 内容 |
|------|------|
| 用法 | 每次 **10-20ml**，含漱 **1-2分钟** 后可吞服或吐掉 |
| 频率 | 每天 **3次**，饭后 |
| 作用 | 促进黏膜再生修复，缩短溃疡愈合天数 |
| 注意 | 味道比较怪，忍一下；和氯己定间隔30分钟以上 |

### 4. 维生素补充

| 补充剂 | 剂量 | 频率 | 作用 |
|--------|------|------|------|
| 维生素C | 500-1000mg | 每天1次 | 促进黏膜修复、免疫支持 |
| 复合维生素B | 1片 | 每天1-2次 | 咽部黏膜修复必需 |

## 四、饮食原则

| ✅ 推荐 | ❌ 避免 |
|---------|---------|
| 冰酸奶（无糖希腊酸奶） | 烫的食物 |
| 蒸蛋（放温凉再吃） | 辣/酸/咸 |
| 凉粥、冷面 | 干硬食物（饼干、坚果） |
| 冰淇淋、布丁 | 橙子、菠萝等酸性水果 |
| 牛奶、奶昔 | 油炸食品 |
| 香蕉、牛油果 | 酒精 |
| 南瓜浓汤（放温） | 含酒精漱口水 |
| 蜂蜜含服（覆盖创面） | |

---

## 五、辅助措施

- 垫高枕头半躺睡（30度），减轻咽部充血
- 床边放温水，醒了润喉
- 洗澡温热水可以，别太烫，10分钟内
- 少说话，让咽部休息
- 冰水小口含着（不喝），局部冷敷减轻肿胀

## 七、就医指征（出现以下任何一条立即去医院）

- 痛感在3天后仍在加重而非减轻
- 出现呼吸困难 / 吞咽困难到连水都咽不下
- 单侧明显肿大 / 张口受限
- 重新高烧（>38.5°C）
- 颈部明显肿胀
- 溃疡面积明显扩大或出现大片坏死
