# 二手房购买与验房清单

> 重点目标：判断房子是否值得买、是否存在重大隐患、后续重新装修会不会产生高额额外成本。

---

## 一、产权与交易风险检查

### 1. 房屋产权
- [ ] 核对房产证 / 不动产权证信息是否与卖方一致。
- [ ] 确认房屋是否有共有产权人，所有产权人是否都同意出售。
- [ ] 确认房屋是否存在抵押、查封、冻结、诉讼纠纷。
- [ ] 确认房屋是否满二 / 满五唯一，影响税费。
- [ ] 确认是否为经济适用房、动迁房、共有产权房等特殊性质房屋。
- [ ] 确认土地性质：住宅、商住、商业等。
- [ ] 核对房屋面积、楼层、用途、地址是否与证件一致。

### 2. 户口与学区
- [ ] 确认原业主户口是否迁出。
- [ ] 在合同中明确户口迁出时间和违约责任。
- [ ] 如果涉及学区，核实学位是否被占用。
- [ ] 向学校、居委、派出所或中介多方确认学区政策。

### 3. 费用欠缴
- [ ] 水费是否结清。
- [ ] 电费是否结清。
- [ ] 燃气费是否结清。
- [ ] 物业费是否结清。
- [ ] 供暖费 / 停车费 / 维修基金是否结清。
- [ ] 宽带、有线电视等是否有欠费或占用端口。

### 4. 合同重点
- [ ] 明确家具家电是否赠送，列清品牌、数量、状态。
- [ ] 明确交房时间。
- [ ] 明确户口迁出时间。
- [ ] 明确房屋质量问题责任。
- [ ] 明确贷款失败、政策变化等情况下的处理方式。
- [ ] 明确违约金比例。
- [ ] 明确物业、水电燃气等交割方式。

---

## 二、房屋结构与安全检查

### 1. 承重结构
- [ ] 查看原始户型图或物业图纸。
- [ ] 确认是否有私自拆改承重墙。
- [ ] 确认阳台配重墙是否被拆除。
- [ ] 确认梁、柱是否被破坏或开槽。
- [ ] 确认门洞是否被擅自扩大。
- [ ] 如果计划重新装修，提前判断哪些墙能拆、哪些墙不能拆。

说明：承重墙问题是红线。涉及结构安全和后续装修审批，严重时会影响整栋楼安全，尽量不要碰。

### 2. 墙面裂缝
- [ ] 检查客厅、卧室、厨房、卫生间墙面是否有裂缝。
- [ ] 检查门窗四角是否有斜向裂缝。
- [ ] 检查梁柱交接处是否有裂缝。
- [ ] 检查墙体是否空鼓、起砂、脱落。
- [ ] 用手敲墙面，听是否有大面积空鼓。

判断：
- 发丝状裂缝：多为腻子层或抹灰层问题，装修时可处理。
- 斜向裂缝、贯穿裂缝：可能涉及沉降或结构问题，要谨慎。
- 大面积空鼓：后续装修需要铲除重做，增加成本。

### 3. 地面与楼板
- [ ] 检查地面是否明显倾斜。
- [ ] 检查地砖、木地板是否空鼓、起翘、松动。
- [ ] 检查楼板是否有明显裂缝。
- [ ] 检查地面是否有返潮、水渍、霉味。
- [ ] 用水平仪或手机水平仪简单测量地面平整度。

### 4. 顶面
- [ ] 检查天花板是否有水渍。
- [ ] 检查顶面是否起皮、发霉、鼓包。
- [ ] 检查顶角是否有裂缝。
- [ ] 如果有吊顶，尽量打开检修口查看内部。

---

## 三、防水与渗漏检查

### 1. 卫生间防水
- [ ] 检查卫生间门口地板是否发黑、起翘。
- [ ] 检查卫生间外侧墙面是否返潮、起皮。
- [ ] 检查墙角是否有霉斑。
- [ ] 检查地漏周边是否积水。
- [ ] 检查马桶底部是否渗水。
- [ ] 检查台盆柜内部是否潮湿、发霉。
- [ ] 如条件允许，要求做闭水试验。

说明：如果重新装修，卫生间通常建议全部拆除，重新做防水和闭水试验。

### 2. 厨房防水与渗漏
- [ ] 检查水槽下方柜体是否发霉、变形。
- [ ] 检查下水管接口是否漏水。
- [ ] 检查厨房墙面、地面是否有水渍。
- [ ] 检查燃气热水器附近是否有渗漏痕迹。

### 3. 外墙与窗边渗水
- [ ] 检查窗台下方是否有水印。
- [ ] 检查窗框四周是否发霉、起皮。
- [ ] 检查外墙内侧是否有潮斑。
- [ ] 雨后看房更容易发现问题。

说明：外墙渗水通常不是自己室内装修就能彻底解决的，可能需要物业或整栋楼维修。

### 4. 顶楼专项
- [ ] 检查顶面是否有渗水痕迹。
- [ ] 检查屋顶防水年限和维修记录。
- [ ] 检查夏季隔热情况。
- [ ] 确认是否有违建、加盖、屋顶公共区域纠纷。

### 5. 一楼专项
- [ ] 检查返潮、霉味、墙角潮湿。
- [ ] 检查下水是否有反水历史。
- [ ] 检查采光是否被绿化或建筑遮挡。
- [ ] 检查蚊虫、噪音、安全性。

---

## 四、水路与排水检查

### 1. 给水系统
- [ ] 打开所有水龙头，检查水压。
- [ ] 同时打开厨房、卫生间水龙头，看水压是否明显下降。
- [ ] 检查冷热水是否正常。
- [ ] 检查水管是否锈蚀、漏水。
- [ ] 检查总水阀是否好用。
- [ ] 了解水管材质：PPR、镀锌管、铝塑管等。

说明：老房子如果仍是老旧镀锌管，装修时建议全屋换管。

### 2. 排水系统
- [ ] 向地漏倒水，检查排水速度。
- [ ] 检查厨房水槽排水是否顺畅。
- [ ] 检查马桶冲水是否有力。
- [ ] 检查是否有反味。
- [ ] 检查下水主管是否老化、生锈、漏水。
- [ ] 了解是否有历史堵塞或反水。

### 3. 管道噪音
- [ ] 听楼上用水时主管道噪音是否明显。
- [ ] 检查下水管是否已做隔音。
- [ ] 如果重新装修，包管时考虑隔音棉。

---

## 五、电路与弱电检查

### 1. 配电箱
- [ ] 查看配电箱是否老旧。
- [ ] 检查空气开关、漏电保护器是否齐全。
- [ ] 检查回路是否清晰标注。
- [ ] 查看是否有烧焦、发黑、异味。
- [ ] 确认入户线容量是否满足现代用电需求。

### 2. 插座与开关
- [ ] 检查每个房间插座数量是否够用。
- [ ] 用插座检测器检查是否接地正常。
- [ ] 检查开关是否松动、打火。
- [ ] 检查卫生间、厨房插座是否有防溅盒。
- [ ] 检查空调、热水器、厨房大功率电器是否有独立回路。

### 3. 老房电路改造判断
- [ ] 房龄超过 10-15 年，优先考虑全屋电路重排。
- [ ] 如果电线老化、线径不足，建议全部更换。
- [ ] 如果计划中央空调、洗碗机、烘干机、蒸烤箱等，需要提前规划电容量。

### 4. 弱电系统
- [ ] 检查宽带入户位置。
- [ ] 检查网线接口是否可用。
- [ ] 检查电视线、电话线是否老旧。
- [ ] 如果要做全屋 Wi-Fi，装修前规划 AP 点位或 Mesh 位置。

---

## 六、燃气与厨房检查

### 1. 燃气安全
- [ ] 检查燃气表位置。
- [ ] 检查燃气管是否锈蚀、私改。
- [ ] 检查燃气阀门是否正常。
- [ ] 确认燃气管道能否改动，以及改动审批流程。
- [ ] 检查燃气热水器安装是否合规。

说明：燃气管道不能随便改，厨房布局设计前必须先确认限制。

### 2. 烟道与排烟
- [ ] 检查厨房公共烟道是否串味。
- [ ] 检查止逆阀是否有效。
- [ ] 检查油烟机排烟是否顺畅。
- [ ] 看烟道附近是否有油污、裂缝、渗漏。

### 3. 厨房布局改造
- [ ] 确认上下水位置。
- [ ] 确认燃气表、燃气管位置。
- [ ] 确认烟道位置。
- [ ] 判断是否适合做开放式厨房。
- [ ] 判断冰箱、洗碗机、蒸烤箱、净水器、垃圾处理器的位置。

---

## 七、门窗与隔音检查

### 1. 入户门
- [ ] 检查门锁是否安全。
- [ ] 检查门框是否变形。
- [ ] 检查开关是否顺畅。
- [ ] 检查密封性和隔音。
- [ ] 如果门太老，装修时考虑更换防盗门。

### 2. 室内门
- [ ] 检查门套是否受潮、变形。
- [ ] 检查门扇是否开裂。
- [ ] 检查合页、门锁是否松动。

### 3. 窗户
- [ ] 检查窗框是否变形。
- [ ] 检查玻璃是否破损。
- [ ] 检查密封胶是否老化。
- [ ] 检查开关是否顺畅。
- [ ] 检查是否漏风、漏水。
- [ ] 临街房重点检查隔音。
- [ ] 老式单层玻璃建议预算内考虑更换断桥铝中空玻璃。

说明：全屋换窗是一笔不小的费用，尤其是大面积窗户或高层施工。

---

## 八、采光、通风与居住体验

### 1. 采光
- [ ] 晴天看真实采光。
- [ ] 阴天看采光下限。
- [ ] 下午看是否西晒严重。
- [ ] 检查低楼层是否被树木、楼栋遮挡。
- [ ] 观察客厅、主卧、厨房、卫生间是否有自然光。

### 2. 通风
- [ ] 打开南北窗，感受是否有穿堂风。
- [ ] 检查卫生间是否有窗。
- [ ] 检查厨房通风是否良好。
- [ ] 暗卫需要重点关注排风系统。

### 3. 噪音
- [ ] 白天听交通噪音。
- [ ] 晚上听邻居、楼道、电梯、水泵噪音。
- [ ] 检查是否靠近主干道、高架、学校、菜场、垃圾房、设备房。
- [ ] 关窗后再判断隔音效果。

### 4. 气味
- [ ] 检查卫生间是否反味。
- [ ] 检查厨房烟道是否串味。
- [ ] 检查楼道是否有垃圾味、潮味。
- [ ] 检查室内是否有霉味或刺鼻装修味。

---

## 九、重新装修前的重点判断

### 1. 是否需要全拆
- [ ] 原装修是否老旧。
- [ ] 水电是否需要全换。
- [ ] 墙地砖是否空鼓。
- [ ] 木地板是否受潮。
- [ ] 吊顶是否发霉或藏污。
- [ ] 厨卫是否需要彻底重做。

### 2. 拆改限制
- [ ] 哪些墙能拆。
- [ ] 哪些墙不能拆。
- [ ] 是否能改卫生间位置。
- [ ] 是否能改厨房位置。
- [ ] 是否能封阳台。
- [ ] 是否能装中央空调 / 新风 / 地暖。
- [ ] 小区物业对装修时间、垃圾清运、材料进场是否有限制。

### 3. 隐蔽工程预算预判
- [ ] 全屋水电重做。
- [ ] 厨卫防水重做。
- [ ] 门窗更换。
- [ ] 墙面铲除到基层。
- [ ] 地面找平。
- [ ] 下水管隔音。
- [ ] 强弱电箱更换。
- [ ] 空调、新风、地暖等设备预留。

### 4. 可能导致预算暴涨的问题
- [ ] 外墙渗水。
- [ ] 顶楼屋面漏水。
- [ ] 一楼反水。
- [ ] 承重结构被破坏。
- [ ] 水电系统严重老化。
- [ ] 下水主管老化漏水。
- [ ] 大面积墙面空鼓。
- [ ] 窗户全部需要更换。
- [ ] 楼上邻居卫生间漏水。

---

## 十、小区与物业检查

### 1. 小区环境
- [ ] 小区绿化是否维护良好。
- [ ] 道路、楼道、电梯是否干净。
- [ ] 垃圾房位置是否影响居住。
- [ ] 是否有明显违建。
- [ ] 停车是否方便。
- [ ] 电动车停放和充电是否规范。

### 2. 物业服务
- [ ] 物业费标准。
- [ ] 物业响应速度。
- [ ] 公共维修是否及时。
- [ ] 电梯维护情况。
- [ ] 安保管理情况。
- [ ] 快递、外卖、访客管理是否方便。

### 3. 邻里情况
- [ ] 楼上是否有小孩、宠物、频繁噪音。
- [ ] 隔壁是否出租房、群租房。
- [ ] 楼道是否堆物。
- [ ] 向邻居打听是否有漏水、停水、停电、返水历史。

## 十一、看房建议

### 1. 建议至少看三次
- [ ] 工作日白天：看采光、噪音、小区状态。
- [ ] 晚上：看噪音、停车、邻居活动。
- [ ] 雨后：看渗水、返潮、排水。

### 2. 建议携带工具
- [ ] 手电筒：看吊顶、柜底、角落。
- [ ] 插座检测器：测插座接线。
- [ ] 卷尺：测尺寸。
- [ ] 手机水平仪：看地面倾斜。
- [ ] 纸巾：擦管道接口看是否渗水。
- [ ] 小瓶水：测试地漏排水。
- [ ] 充电器：测试插座通电。
- [ ] 口罩：进入潮湿、霉味明显的房间。

### 3. 看房时不要只看表面
- [ ] 不要被新刷墙面迷惑。
- [ ] 不要被软装和灯光迷惑。
- [ ] 重点看墙角、窗边、柜底、管道、吊顶内部。
- [ ] 如果房子刚刚简单翻新过，要警惕“串串房”。

---

## 十二、简单决策标准

### 可以重点考虑
- [ ] 产权清晰。
- [ ] 无重大结构问题。
- [ ] 无严重渗漏。
- [ ] 小区物业稳定。
- [ ] 采光通风良好。
- [ ] 户型适合改造。
- [ ] 装修成本可控。

### 需要谨慎
- [ ] 老房水电严重老化。
- [ ] 墙面、顶面多处水渍。
- [ ] 下水管道老旧。
- [ ] 噪音明显。
- [ ] 物业管理差。
- [ ] 一楼潮湿或有返水风险。
- [ ] 顶楼有漏水历史。

### 尽量避开
- [ ] 承重墙被破坏。
- [ ] 产权复杂或有纠纷。
- [ ] 查封、抵押未解决。
- [ ] 外墙严重渗水且物业不处理。
- [ ] 楼上长期漏水未解决。
- [ ] 小区存在严重群租、违建、安全隐患。
- [ ] 房屋存在明显结构沉降或贯穿裂缝。

## 十四、一句话提醒

二手房最怕的不是旧，而是：产权不清、结构动过、漏水严重、下水反水、外墙渗水、物业不管。  
如果只是装修旧、水电老、墙面差，只要价格合适，反而可以通过重新装修解决。
